# AI起名工具 MVP功能清单与分期排期

> 编制日期：2026-06-12 | 版本：V1.0
> 技术路线：方案C（买源码 + 替换引擎 + 自研AI层）

---

## 一、MVP定义与原则

### 1.1 MVP要验证的核心假设

| 假设 | 验证方式 | 成功标准 |
|------|---------|---------|
| **用户愿意为AI八字起名付费** | 上线付费按钮，统计转化率 | 自然流量下付费转化率 ≥ 3% |
| **八字排盘准确性可被用户感知** | 埋点统计用户查看命理报告的深度 | 报告完读率 ≥ 60% |
| **AI生成的五行匹配名字质量过关** | 收集用户收藏/复制行为 | 单次会话平均收藏 ≥ 2个名字 |
| **合规定位"传统文化工具"能通过审核** | 小程序/H5提交应用商店审核 | 一次过审，无违规下架 |

### 1.2 MVP成功标准

- ✅ 完整跑通「输入出生信息 → 八字排盘 → 五行分析 → 名字生成 → 结果展示」闭环
- ✅ 排盘准确性 ≥ 95%（与专业命理师核对100个样本）
- ✅ 单次起名从输入到出结果 ≤ 30秒
- ✅ 上线后2周内自然流量获取 ≥ 500次使用
- ✅ 0监管投诉 / 0违规下架

### 1.3 MVP原则

1. **先准后全**：排盘准确性是第一生命线，宁缺功能不缺精度
2. **买来即用**：八字引擎用开源库、汉字库用互站网源码，不自研底层
3. **合规兜底**：所有页面标注"仅供传统文化参考"，不碰"算命改运"红线
4. **1人1周**：以方案C为基准，单人7个工作日出MVP
