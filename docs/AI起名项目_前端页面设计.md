## 一、信息架构

### 1.1 站点地图

```
首页（八字排盘+起名入口）
├── 八字起名流程
│   ├── Step 1: 基本信息（姓名/性别/出生日期时间/出生地）
│   ├── Step 2: 排盘结果（四柱八字+五行分布+十神）
│   ├── Step 3: 喜用神分析（格局→用神→喜神→忌神）
│   ├── Step 4: 起名偏好（姓氏/字数/风格/诗词出处偏好）
│   ├── Step 5: AI生成名字（流式输出+名字列表）
│   └── Step 6: 名字详情（单字解读+五行补益+诗词出处+评分）
│
├── 测名评分
│   ├── 输入名字（姓+名+出生信息）
│   └── 评分结果（五行匹配度+三才五格+音韵+重名风险+谐音检测）
│
├── 我的
│   ├── 起名历史
│   ├── 收藏的名字
│   ├── 家庭投票
│   ├── 分享记录
│   ├── 设置（暗色模式/隐私/关于）
│   └── 付费中心（会员/单次解锁）
│
└── 通用页面
    ├── 启动页/引导页
    ├── 付费解锁弹窗
    ├── 分享卡片（微信/朋友圈）
    └── 关于/免责声明
```

### 1.2 导航结构

**主入口（底部Tab）**：
| Tab | 图标 | 路由 | 说明 |
|-----|------|------|------|
| 首页 | Home | `/` | 八字排盘+起名主流程 |
| 测名 | Search | `/test-name` | 测名评分 |
| 我的 | User | `/profile` | 个人中心 |

**子页面导航**：顶部导航栏，标题居中，左侧返回，右侧操作按钮

## 三、核心页面设计

### 3.1 P01 启动页

**目的**：品牌第一印象，传达"传统文化数字化工具"定位

**布局**：
```
┌─────────────────────┐
│                     │
│                     │
│     [品牌Logo]      │  ← 极简Logo，靛青色
│                     │
│    "名字，有来处"    │  ← 品牌Slogan（思源宋体）
│                     │
│  传统文化数字化工具   │  ← 产品定位说明
│                     │
│                     │
│   ┌───────────┐     │
│   │  开始起名  │     │  ← Primary Button
│   └───────────┘     │
│                     │
│  已有名字？测一测 →  │  ← Text Button
│                     │
│  ───────────────── │
│  首次使用表示同意    │
│  《用户协议》《隐私》 │
└─────────────────────┘
```

**设计要点**：
- 大量留白，视觉焦点在Logo和Slogan
- 无装饰图案，用留白传达高级感
- 底部合规信息必须在首屏可见
- 进入动画：Logo淡入300ms → Slogan淡入300ms（延迟200ms）→ 按钮滑入

**品牌Slogan备选**：
1. "名字，有来处"（推荐——文化感+故事感）
2. "每个名字，都有出处"（直白+诗意）
3. "好名字，有道理"（双关——道理=道理+道法理）

│  └──────┘  └──────┘        │
│                             │
│  出生日期 *                  │
│  ┌───────────────────────┐  │

│  └───────────────────────┘  │

**设计要点**：
- 步骤指示器始终可见，让用户知道进度
- 出生日期/时间使用原生 picker 组件（体验最优）
- 出生地用于真太阳时校正，标注"选填但推荐"
- "不确定具体时辰"→ 勾选后时间选择器置灰，分析时按12时辰综合取用
- 表单校验：姓氏必填（1-2字），日期必填，时间必填（或勾选不确定）
- 键盘适配：时间选择器弹起时，内容区域不上推（避免布局跳动）

│  │                       │  │

**设计要点**：
- 命盘组件是全产品最核心的组件，必须做到专业+清晰
- 四柱表使用等宽字体，天干地支竖排对齐
- 五行色块用设计规范中的五行专属色，视觉辨识度高
- 喜用神区域是差异化核心——"不是缺啥补啥"的提示必须醒目
- 点击任一柱 → 底部面板展开该柱详细分析（藏干解读+十神+冲合）
- 首次进入排盘页时，四柱逐列淡入（动画见设计规范动效章节）

**付费门槛设计**：
- 基本排盘+五行分布：免费展示
- 喜用神详细分析 + 格局解读：需付费解锁（模糊遮罩+解锁按钮）
- 免费用户可见：格局名称 + 喜用神名称，但详细解读文字模糊处理
