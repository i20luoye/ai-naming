## 一、产品概述

### 1.1 产品定义
基于八字命理学的AI智能起名微信小程序，用户输入出生信息后，系统自动完成八字排盘、五行分析、喜用神判定，并生成匹配命理的精选名字及详细解读。

### 1.2 核心差异化
- **透明化解题**：展示完整的八字分析推导过程，而非黑箱输出结果
- **精准喜用神**：基于日主强弱+五行生克的算法判定，超越竞品"缺啥补啥"
- **双人协作**：夫妻共同参与起名（第二阶段）

### 1.3 平台
- MVP：微信小程序
- 后续：Web端、App（Flutter跨平台）

## 四、算法逻辑说明

### 4.1 八字排盘

**输入**：出生年月日时（阳历/农历）+ 出生地经纬度  
**处理**：
1. 公历转农历（如输入为阳历）
2. 计算真太阳时（基于出生地经度校准）
3. 确定年柱（以立春为界）
4. 确定月柱（以节气为界，正月寅月）
5. 确定日柱（干支纪日计算）
6. 确定时柱（时辰对应地支+日干定天干）

**输出**：四柱八字 + 五行分布 + 日主天干

### 4.2 喜用神判定

**输入**：四柱八字 + 五行分布  
**处理**：
1. 日主强弱分析
   - 得令（月令是否生扶日主）
   - 得地（日支是否生扶日主）
   - 得势（其他干支是否生扶日主）
2. 综合判定：身强 / 身弱 / 中和
3. 喜用神确定
   - 身弱 → 喜生扶日主的五行（印星、比劫）
   - 身强 → 喜克制日主的五行（官杀、食伤、财星）
   - 中和 → 喜通关调候的五行

**输出**：喜神 / 用神 / 忌神 / 闲神 列表

### 4.3 AI名字生成

**输入**：喜用神 + 姓氏 + 性别 + 风格偏好  
**处理**：
1. 规则引擎筛选：从字库中筛选符合喜用神五行的汉字
2. LLM生成：基于筛选字库 + 风格偏好生成候选名字
3. 规则引擎校验：
   - 五格数理检查（天格/人格/地格/外格/总格）
   - 三才配置检查
   - 五行匹配度打分
   - 谐音/不雅联想过滤
4. 排序输出：综合评分排序

**输出**：20个精选名字 + 每个名字的完整解析
