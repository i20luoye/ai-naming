## 五、关键结论

1. **八字排盘：不需要自研。** `lunar-python`（MIT）是行业标准，直接拿来用。它已被多个项目交叉验证，准确率100%。

2. **喜用神分析：有现成算法可参考。** `korean-saju`（MIT）是唯一实现了用神推导的开源库。虽然是韩文，但代码逻辑通用，可以翻译/适配。这是我们的核心壁垒——90%竞品做不到。

3. **起名逻辑：有参考但需要自研。** `CrystalMarch/bazi`（MIT）是唯一把八字+起名放一起的项目，可以参考其五行字库和架构，但代码太老需要重写。

4. **架构蓝图：Destiny插件思路。** "底层确定性计算 + 上层LLM创意生成"是最优架构。计算交给开源库，创意交给大模型。

5. **最小可行路径**：
   - 第1步：`pip install lunar-python` → 跑通八字排盘（1天）
   - 第2步：参考 `korean-saju` 实现喜用神判断（3-5天）
   - 第3步：构建五行汉字库 + LLM名字生成prompt（2-3天）
   - 第4步：包装成API + 小程序前端（1周）
   - **总计：2-3周可出MVP**
