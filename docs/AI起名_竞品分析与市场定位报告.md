## 一、执行摘要

### 1.1 一句话结论

**市场上不缺起名工具，但缺一个真正懂「喜用神」的AI起名师——这是90%竞品的算法盲区，也是我们的核心机会。**

### 1.2 关键判断

| 维度 | 结论 | 置信度 |
|------|------|--------|
| 赛道是否成立 | ✅ 成立。年市场~45亿，增速25-30% | 高 |
| 竞品是否强大 | ⚠️ 产品多但专业度普遍低 | 高 |
| 差异化空间 | ✅ 明确。喜用神分析是90%竞品盲区 | 高 |
| 技术可行性 | ✅ 开源生态成熟，MVP 2-3周 | 高 |
| 盈利路径 | ✅ Freemium → 付费报告 → 人工定制 → B2B API | 中高 |
| 主要风险 | 命理争议、监管风险、LLM成本 | — |

### 1.3 建议决策

**立即启动MVP开发。** 市场窗口存在，竞品护城河不深，开源基础设施成熟。先做"喜用神分析+AI起名"这个核心功能验证，再扩展。

## 二、市场全景

### 2.1 市场规模测算

| 细分市场 | 估算逻辑 | 市场空间 |
|----------|----------|----------|
| 宝宝起名 | 年新生儿~900万 × 10%渗透率 × ¥50客单价 | ~4.5亿/年 |
| 成人改名 | 存量需求，年约200万人 × ¥100客单价 | ~2亿/年 |
| 企业/品牌命名 | 年新注册企业~3000万 × 0.5% × ¥300 | ~4.5亿/年 |
| 店铺/产品命名 | 长尾需求 | ~2亿/年 |
| **合计（C端可触达）** | | **~13亿/年** |
| 含B2B API输出 | 向母婴/婚恋/风水平台输出 | +5-10亿/年 |

> 来源：博研咨询《2024-2030中国AI企业名称生成器行业现状及趋势》等公开报告。C端渗透率取保守值。

### 2.2 市场增速

- 2024年AI起名市场约35亿 → 2025年预计45亿，增速~28%
- 驱动因素：AI普及降低使用门槛、年轻父母对传统文化+科技结合的接受度提升、微信小程序生态成熟
- 增速预计在未来3年保持25%+，之后趋于平稳

### 2.3 竞争格局（梯队划分）

```
第一梯队（头部，市占率合计~40%）
├── 千千起名        ★★★★★  600万+用户  全能型
└── 周易起名大师    ★★★★    老牌产品    工具型

第二梯队（腰部，市占率合计~30%）
├── 八字测测        ★★★★    B2B为主    源码交付
├── 全能起名神器    ★★★     中小用户    流量型
└── 起名通/宝宝起名类 ★★★   细分场景    场景型

第三梯队（长尾，市占率合计~30%）
├── 诗词起名类      ★★★     文化向      小众
├── 五行取名类      ★★      极简向      免费
├── 手工起名师傅    线下     高客单      不可规模化
└── 闲鱼AI起名服务  个体     灵活        质量参差
```

### 2.4 行业关键趋势

1. **从"经验直觉"到"代码体系"**：传统命理正在被工程化，可验证、可迭代
2. **微信小程序是主要分发渠道**：搜索"起名"自然流量巨大
3. **"源码交付"模式兴起**：B端客户买源码自建品牌运营
4. **API生态初步形成**：gugudata、apizero等已提供八字起名API
5. **年轻化趋势明显**：95后父母更接受AI起名，但对"迷信"标签敏感

## 四、头部竞品深度拆解

### 4.1 千千起名 — 最强竞品

**基本信息**
- 开发方：天津企航志成科技有限公司
- 定位：「AI+国学全能起名平台」
- 累计用户：600万+
- 覆盖场景：宝宝起名、成人改名、商业命名

**核心竞争力**
1. **真太阳时+经纬度校准排盘**：比只用北京时间的竞品精准
2. **喜用神精准定位**：不是"缺啥补啥"，是真正分析日主强弱+调候
3. **全维度校验**：三才五格 + 周易卦象 + 生肖宜忌 + 重名筛查 + 方言谐音
4. **百万级国学字库**：诗经、楚辞、唐诗、宋词全覆盖
5. **永久免费策略**：八字测算、五行分析、智能起名、打分、重名查询全部免费

**盈利模式**
- 基础功能永久免费（获客引擎）
- 人工定制服务（大师一对一，明码标价）
- 新用户送3个专属名字（体验转化）

**我们的可攻击弱点**
1. **品牌调性老化**："国学大师"风，对95后父母吸引力不足
2. **AI感不足**：虽然叫"AI起名"，但交互偏传统表单，缺少现代AI产品的对话感
3. **解释系统不透明**：告诉你"这个名字好"，但推理链路不清晰
4. **缺少反向验证**：只说好不说坏，没有"魔鬼辩护"机制
5. **缺少社区/内容生态**：纯工具，没有UGC、案例分享、口碑传播
6. **无API开放**：没有B2B能力，生态位单一

### 4.2 周易起名大师 — 老牌工具

**基本信息**
- 定位：传统周易起名软件
- 核心卖点：10万+汉字组合库、机器学习优化、跨平台同步
- 声称准确率：五行匹配98.7%

**优势**
- 品牌认知度较高（老牌产品）
- 买断制，用户决策成本低
- 跨平台（PC+手机）

**弱点**
- **AI成分少**：本质是传统算法+数据库，非真正AI
- **品牌老化**：对年轻用户吸引力下降
- **缺少社区/内容**：纯工具，用户用完即走
- **无人工定制**：无法做高客单价

### 4.3 八字测测 — B2B模式

**定位**：AI八字起名系统源码交付
**模式**：卖源码给B端客户，客户自建品牌运营

**对我们的启示**
- B2B是验证可行的商业模式
- 源码交付说明八字+起名的技术门槛已被工程化
- 但源码交付模式做的是"卖铲子"，不碰C端——我们的差异化在C端体验

### 4.4 闲鱼AI起名 — 个体户模式

**模式**：个人在闲鱼接单，AI辅助起名
**定价**：68-128元/单
**收入**：月入3000-10000元

**对我们的启示**
- **需求真实且付费意愿强**：有人愿意为AI起名付费
- **个体户已验证PMF**：说明产品-市场匹配存在
- **AI起名的"手工作坊"阶段**：正好是我们用产品化替代的机会

## 六、竞品盈利模式拆解

### 6.1 四种已验证的盈利模型

#### 模型A：Freemium + 人工定制（千千起名）
```
免费八字排盘 + 五行分析 + 3个名字
          ↓ 转化率 3-5%
¥9.9-99 完整起名报告 + 详细命理分析
          ↓ 转化率 1-2%
¥200-2000 人工大师复核 + 定制方案
```
**优点**：漏斗清晰，高客单有支撑
**缺点**：人工定制难以规模化

#### 模型B：一次性买断（周易起名大师）
```
¥30-100 永久使用
```
**优点**：用户决策快，现金流好
**缺点**：LTV低，缺少持续收入

#### 模型C：B2B源码/API（八字测测、gugudata）
```
¥5000-20000 源码交付 / 按调用量计费
```
**优点**：客单价高，规模化容易
**缺点**：市场天花板有限，依赖渠道

#### 模型D：广告变现（全能起名神器）
```
免费工具 + 信息流广告
```
**优点**：零门槛获客
**缺点**：ARPU极低，需要海量用户

### 6.2 推荐盈利组合（我们的）

```
层级        产品                  定价         目标转化率
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
L1 引流     免费八字分析+3个名字    ¥0           100%（获客）
L2 转化     完整起名报告            ¥9.9/次      5-8%
L3 高客单   人工大师定制            ¥99-299/次   1-2%
L4 B2B      API输出              按调用量      渠道合作
L5 增值     改名/合婚/择日          ¥6.9-19.9   交叉销售
```

**与竞品的差异**：
- L2定价¥9.9（低于千千起名的人工定制¥200+），降低决策门槛
- L3人工定制做"AI辅助+大师审核"（而非纯人工），降本增效
- L4 B2B API是千千起名没有的增量市场
- L5 增值功能矩阵（合婚/择日/运势）延长用户LTV

| 空白点 | 现状 | 机会大小 | 难度 |
