# AI起名工具 — 产品需求文档（PRD 合并版）

> **版本**：v2.0（合并版） | **日期**：2026-06-12 | **作者**：产品团队

> **合并说明**：本文档以产品经理PRD为战略骨架，植入产品运营PRD的执行层设计，实现「战略到执行全覆盖」

## 一、产品概述

### 1.1 一句话定位

**一款以「八字五行精准排盘 + 汉字文化深度匹配」为核心、AI大模型增强体验的智能起名工具。**  
定位为「AI起名工具」（文化解读 + 音形义筛选），**不是「AI算命」**。

产品Slogan：**「AI八字起名，看懂再选」**

### 1.2 目标用户

| 用户群 | 占比（预估） | 核心场景 |
|--------|:--------:|---------|
| 新生儿父母（25-35岁） | 60% | 为新生儿取一个好名字，兼顾八字五行补缺、音韵美感、文化寓意 |
| 成人改名用户 | 25% | 对现有名字不满意，希望根据八字重新匹配一个更合适的名字 |
| 海外华人 | 15% | 身在海外，希望通过在线工具获得传统文化起名服务，解决时区差异问题 |

### 1.3 核心价值主张

| 痛点 | 我们的解决方案 |
|------|--------------|
| 现有AI起名排盘不准（大模型直接排盘错误率40%） | **规则引擎排盘**，cnlunar/lunar-typescript 保证100%准确 |
| 「缺啥补啥」过于机械，未区分喜用神与五行缺失 | **五行旺衰 + 喜用神推断**，精准定位真正需要补的五行 |
| 名字同质化严重（梓涵、一诺泛滥） | **60,000条康熙字典数据库** + AI文化释义，生成差异化名字 |
| 解读浅层，缺乏文化深度 | **大模型深度解读**：诗词典故、字形字义、音韵分析 |
| 海外华人无法享受专业服务 | **真太阳时校正** + 全线上服务，覆盖全球时区 |

### 1.4 产品差异化定位

```
┌──────────────────────────────────────────────────┐
│           竞品：大模型直接起名                      │
│   输入 → 大模型 → 名字（排盘40%错误率）             │
│   同质化严重，解读 = 话术拼接                       │
└──────────────────────────────────────────────────┘
                        vs
┌──────────────────────────────────────────────────┐
│        本产品：规则引擎 + AI增强（混合架构）         │
│   输入 → 规则引擎排盘（100%准确）→ 汉字匹配引擎     │
│        → AI文化深度解读 → 个性化名字推荐            │
│   精准排盘 + 深度解读 + 差异化的名字                 │
└──────────────────────────────────────────────────┘
```

### 1.5 平台

- MVP：微信小程序
- 后续：Web端（H5）、App（Flutter跨平台）

## 四、页面详细设计

### 4.1 功能架构总览

```
AI八字起名小程序
├── 首页
│   ├── 起名入口（核心CTA）
│   ├── 名字灵感（每日推荐）
│   └── 我的起名记录
├── 信息输入页
├── 八字排盘结果页
├── 喜用神分析页（付费墙在此）
├── 名字列表页（付费后完整版）
├── 名字详情页
├── 收藏与对比页
├── 分享卡片
└── 个人中心
```

### 4.2 首页

**目标**：3秒内让用户理解产品价值，引导进入起名流程

**页面元素**：
- 顶部：产品Slogan「AI八字起名，看懂再选」
- 中间：大号CTA按钮「免费测八字，开始起名」
- 下方：3个信任背书卡片（轮播）
  - 「精准八字排盘，真太阳时校准」
  - 「完整分析过程，每一步都看得懂」
  - 「百万字库精选，每个名字有出处」
- 底部：今日推荐名字（3个，带五行标签，吸引点击）
- 底部导航栏：首页 / 起名 / 收藏 / 我的

**交互**：
- 点击CTA → 进入信息输入页
- 点击推荐名字 → 进入名字详情页
- 信任卡片自动轮播，支持手动滑动

### 4.3 信息输入页

**目标**：引导用户完整、准确地输入出生信息

**表单字段**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|:----:|------|
| 宝宝性别 | 单选按钮 | 是 | 男/女，影响名字风格 |
| 出生日期 | 日期选择器 | 是 | 公历/农历切换 |
| 出生时间 | 时间选择器 | 是 | 精确到分钟，标注"越准确越好" |
| 出生地 | 省-市-区三级选择 | 是 | 用于真太阳时校准 |
| 姓氏 | 文本输入 | 是 | 限1-2个汉字 |
| 是否已出生 | 开关 | 是 | 未出生则提示"预产期仅供参考" |

**交互**：
- 所有字段填写完成后，底部按钮亮起「开始八字排盘」
- 未出生模式开启时，出生日期标签改为"预产期"，底部出现黄色提示条
- 字段按顺序引导填写，当前字段完成后自动聚焦下一个

**校验规则**：
- 出生日期不晚于今天（已出生）/ 不早于今天前300天（预产期）
- 出生时间 00:00-23:59
- 姓氏仅限中文汉字，1-2个字
- 所有必填项完成后按钮才可点击
- 未来日期直接拒绝，弹出明确提示

### 4.4 八字排盘结果页

**目标**：用可视化方式展示八字排盘结果，让用户"看懂"

**页面结构**（从上到下）：

1. **四柱八字卡片**（视觉重点）
   ```
   ┌──────────────────────────────┐
   │   年柱    月柱    日柱    时柱  │
   │   甲子    乙丑    丙寅    丁卯  │
   │   木水    木土    火木    火木  │
   └──────────────────────────────┘
   ```

2. **五行雷达图**
   - 金/木/水/火/土 五维分布
   - 直观展示五行强弱，动画效果呈现

3. **日主分析**
   - 日主天干（如「丙火」）
   - 日主强弱判定（身强/身弱/中和）
   - 通俗解释：「您的宝宝日主为丙火，生于丑月，火气偏弱...」

4. **继续按钮** → 进入喜用神分析（付费墙在此）

**交互**：
- 八字卡片支持长按保存图片
- 五行雷达图带动画效果
- 日主分析区域支持展开/收起

### 4.5 喜用神分析页（付费转化点）

**目标**：展示专业分析过程，建立信任，引导付费

**页面结构**：

1. **日主强弱详细分析**
   - 得令/得地/得势三维分析
   - 每项有得分和通俗解释

2. **五行喜忌结论**
   ```
   ✅ 喜神：木、火（补益日主，增强运势）
   ⚠️ 忌神：金、水（克制日主，需避免）
   ➖ 闲神：土（影响较小）
   ```

3. **起名原则**
   - 「名字中应多选用属木、属火的汉字」
   - 「避免属金、属水的字」

4. **付费墙**
   - 上方展示前3个名字的预览（名字+五行标签，不可点详情）
   - 付费按钮：「解锁全部20个精选名字 + 详细解析报告  ¥98」
   - 下方小字：「一次付费，永久查看」

**付费后**：展开完整名字列表 + 解锁名字详情

### 4.6 名字列表页（付费后完整版）

**目标**：让用户高效浏览和筛选名字

**列表展示**：
- 每行一个名字，显示：
  - 名字（大号字体）

## 五、算法逻辑

### 5.1 八字排盘算法

**输入**：出生年月日时（公历/农历）+ 出生地经纬度

**处理步骤**：
1. 公历转农历（如输入为公历）
2. 计算真太阳时（基于出生地经度校准）
3. 确定年柱（以立春为界）
4. 确定月柱（以节气为界，正月寅月）
5. 确定日柱（干支纪日计算）
6. 确定时柱（时辰对应地支 + 日干定天干）

**输出**：四柱八字 + 五行分布 + 日主天干

### 5.2 喜用神判定算法

**输入**：四柱八字 + 五行分布

**处理步骤**：
1. **日主强弱分析**
   - 得令（月令是否生扶日主）
   - 得地（日支是否生扶日主）
   - 得势（其他干支是否生扶日主）
2. **综合判定**：身强 / 身弱 / 中和
3. **喜用神确定**
   - 身弱 → 喜生扶日主的五行（印星、比劫）
   - 身强 → 喜克制日主的五行（官杀、食伤、财星）
   - 中和 → 喜通关调候的五行

**输出**：喜神 / 用神 / 忌神 / 闲神 列表

### 5.3 AI名字生成流程

**输入**：喜用神 + 姓氏 + 性别 + 风格偏好（可选）

**处理步骤**：
1. **规则引擎筛选**：从60,000条康熙字典字库中筛选符合喜用神五行的汉字
2. **LLM生成**：基于筛选字库 + 风格偏好生成候选名字组合
3. **规则引擎校验**：
   - 五格数理检查（天格/人格/地格/外格/总格）
   - 三才配置检查
   - 五行匹配度打分
   - 谐音/不雅联想过滤
4. **排序输出**：综合评分排序

**输出**：20个精选名字 + 每个名字的完整解析
