## 一、品牌调性定义

### 1.1 品牌关键词

| 维度 | 关键词 | 说明 |
|------|--------|------|
| 气质 | 温润、克制、通透 | 不是"玄学大师"的压迫感，是"文化顾问"的专业感 |
| 态度 | 严谨、透明、善意 | 不故弄玄虚，推理过程全透明 |
| 视觉 | 现代极简、东方意蕴 | 当代设计语言承载传统文化内容 |
| 情感 | 温暖、安心、尊重 | 让用户感到"被认真对待" |

### 1.2 品牌人格画像

- **不是**：穿着道袍摇着蒲扇的算命先生
- **而是**：穿着素色衬衫、戴着细框眼镜、书架上放着《协纪辨方书》和《人间词话》的文化顾问

### 1.3 品牌语调规范

| ✅ 可以说 | ❌ 不可以说 | 原因 |
|----------|-----------|------|
| 五行分析 | 五行算命 | 合规红线 |
| 喜用神解读 | 算你的喜用神 | 避免预测暗示 |
| 传统文化参考 | 改运/转运 | 合规红线 |
| 名字寓意分析 | 名字吉凶预测 | 避免迷信暗示 |
| 命理文化解读 | 命中注定 | 避免决定论 |
| 建议参考 | 必须这样 | 保持中立客观 |
| AI智能分析 | 大师测算 | 避免虚假权威 |

### 1.4 与竞品的视觉差异化

| 竞品通用风格 | 我们的方向 |
|-------------|-----------|
| 暗色背景+金色字体（"玄学"风） | 浅色为主+温润点缀色（"文化"风） |
| 龙凤祥云纹样装饰 | 留白+几何线条+极少量水墨元素 |
| 大字标题+闪烁动效 | 信息层级清晰+克制动效 |
| "大师"人像背书 | AI标识+数据可视化+推理链透明 |
| 红色/金色为主色调 | 靛蓝/松石绿+暖灰为主色调 |

### 2.2 字体系统

#### 字体选型

| 用途 | 中文 | 西文 | 说明 |

```css
/* 仅加载用到的字重 */
@import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600&family=Playfair+Display:wght@500;700&display=swap');
```

#### 特殊排版规范

**干支排列**：四柱八字的天干地支使用等宽字体 `JetBrains Mono`，保证各柱对齐：
```
年柱    月柱    日柱    时柱
庚午    辛巳    甲子    丙寅
```

**五行色块**：每个五行属性前加 8×8px 圆角色块，色值对应五行专属色。

### 2.3 图标系统

- **图标库**：Lucide Icons（MIT协议，轻量，线条风格统一）
- **线条粗细**：1.5px（与文字视觉重量匹配）
- **尺寸体系**：16 / 20 / 24 / 32 px
- **视觉风格**：线性图标，圆角2px，不使用填充图标
- **自定义图标**：五行元素（金木水火土符号）需定制设计，风格与Lucide一致

### 2.4 插画风格

- **风格**：极简线条+局部淡彩，类似传统白描手法
- **场景**：仅用于空状态、引导页、结果页顶部装饰
- **色彩**：使用主色+辅色的淡色变体（opacity 20%-40%）
- **禁忌**：不使用卡通人物、不使用道教符咒图案、不使用龙凤祥云等传统吉祥纹样

## 五、动效规范

### 5.1 动效原则

- **有目的**：每个动效都要服务于信息传达，不为炫技
- **克制**：时长控制在200-400ms，缓动函数优先使用 ease-out
- **物理感**：符合真实物理直觉（加速→减速）

### 5.2 缓动函数

| Token | 值 | 用途 |
|-------|----|------|
| `--ease-default` | `cubic-bezier(0.25, 0.1, 0.25, 1)` | 通用过渡 |
| `--ease-in` | `cubic-bezier(0.4, 0, 1, 1)` | 元素退出 |
| `--ease-out` | `cubic-bezier(0, 0, 0.2, 1)` | 元素进入 |
| `--ease-spring` | `cubic-bezier(0.34, 1.56, 0.64, 1)` | 弹性效果（收藏心跳等） |

### 5.3 时长规范

| 场景 | 时长 | 缓动 |
|------|------|------|
| 按钮hover/active | 150ms | default |
| 弹窗/面板弹出 | 300ms | ease-out |
| 弹窗/面板关闭 | 200ms | ease-in |
| 页面切换 | 300ms | ease-out |
| 列表项展开/折叠 | 250ms | ease-out |
| 数字递增动画 | 800ms | ease-out |
| AI流式打字 | 30-50ms/字 | — |
| 五行色块脉冲 | 1500ms | ease-in-out (infinite) |
| 收藏心跳 | 400ms | spring |

### 5.4 关键动效描述

**八字排盘结果展开**：
1. 四柱天干地支依次淡入（每柱延迟100ms）
2. 五行色块从中心缩放出现（duration 300ms）
3. 藏干区域折叠态显示，点击展开（高度过渡 250ms）

**AI分析报告流式输出**：
1. 左侧竖线开始脉冲
2. 文字逐字出现（30-50ms/字），模拟打字
3. 段落完成后竖线移至下一段开头
4. 全部输出完成 → 竖线消失 → 出现AI标识

**名字卡片出现**：
1. 卡片从下方偏移20px + opacity 0 过渡到原位 + opacity 1
2. 每张卡片延迟80ms（瀑布流效果）
3. 五行色块延迟200ms后弹出（spring缓动）
